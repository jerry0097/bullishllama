About Bullish Llama

Bullish Llama is a technically-driven financial decision support system built with a dual-layer strategy: combining real-time stock market data with AI-enhanced fundamental and technical analysis. The platform is designed to assist individual investors in making informed and balanced portfolio decisions based on their investment amount and risk appetite.

Core Features
  • Real-Time Data Integration via Dhan APIs, supporting both bluechip and penny stock screening.
  • Heuristic-Based Technical Analysis: Determines buy signals (Buy / Strong Buy / Neutral / Wait) based on stock price positions relative to 1-year high/low trends.
  • LLM-Driven Fundamental Analysis using Gemini API to provide intelligent investment commentary and recommendations.
  • Dynamic Portfolio Allocation Engine that factors in both price thresholds and diversification constraints across Low, Medium, and High risk appetites.
  • Responsive Frontend using Flask + MaterializeCSS with custom UI components and validation logic.
Technical Stack
  • Backend: Python (Flask)
  • Data Sources: Dhan Stock Screener API, Google Gemini LLM API
  • Frontend: MaterializeCSS + HTML5
  • Deployment: Localhost (Development), Future-ready for Docker/Cloud
